Optional key

key?: number
  • Defined in nbs/interfaces/note/NoteOptions.ts:8

The key of the note block, from 0-87, where 0 is A0 and 87 is C8. 33-57 is within the 2-octave limit.

Optional velocity

velocity?: number
  • Defined in nbs/interfaces/note/NoteOptions.ts:13

The velocity/volume of the note block, from 0% to 100%.

Optional panning

panning?: number
  • Defined in nbs/interfaces/note/NoteOptions.ts:18

The stereo position of the note block, from 0-200. 100 is center panning.

Optional pitch

pitch?: number
  • Defined in nbs/interfaces/note/NoteOptions.ts:23
